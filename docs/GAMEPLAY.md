# 🎮 像素小屋交互指南

## 房间总览

进入网站后，你会看到一间像素风格的紫色调小屋。房间里的每个物件都可以点击互动。

## 🏠 物件交互说明

| 物件 | 位置 | 功能 | 说明 |
|------|------|------|------|
| 📚 书架 | 左上方 | 博客日记 | 点击打开博客文章列表，查看/发布笔记 |
| 👗 衣橱 | 左下方 | 衣服商城 | 用金币购买新衣服，切换角色穿搭 |
| 🚪 门 | 中上方 | 施工中 | 点击后显示「门外的世界正在施工中...」气泡 |
| 🖼️ 画板 | 右上方 | 访客留言板 | 登录用户可以在画板上留言 |
| 🧙‍♀️ 小屋主人 | 中央偏下 | 个人信息 | 点击查看博主身份信息界面 |
| 🐱 小猫 | 右下方 | 正在睡觉 | 点击进入小猫睡觉的全屏动画界面 |
| 🛏️ 床铺 | 右侧 | 小游戏 | 点击打开日历/游戏入口 |
| 🪟 窗户 | 中右上方 | 窗外风景 | 点击欣赏窗外的像素风景动画 |

## 📚 书架 — 博客系统

- 普通访客：浏览文章、发表评论、点赞
- 博主身份：新增/编辑/删除文章
- 支持 Markdown 格式书写
- 楼中楼评论系统

## 👗 衣橱 — 换装系统

- 初始金币：200 🪙
- 默认拥有「学院校服」
- 可购买的服装：
  - ⚔️ 冒险者斗篷（100 金币）
  - 🌙 睡衣（60 金币）
  - 🔮 女巫魔法袍（150 金币）
- 购买后可随时切换穿戴

## 🖼️ 画板 — 留言板

- 需要注册登录后才能留言
- 每条留言最多 200 字
- 显示留言者头像、昵称和时间

## 🐱 小猫

- 点击小猫会进入全屏睡觉画面
- 小猫目前在睡觉，暂时无法互动
- 点击「返回小屋」可回到主页面

## 🪟 窗户

- 展示窗外的像素风景
- 有动态的云朵飘动效果
- 太阳、树木组成宁静的户外场景

## 🛏️ 床铺

- 打开日历视图
- 可查看和切换月份
- 高亮显示今天的日期

## 💡 操作提示

1. 底部有快捷按钮栏，可以直接点击对应功能
2. 右上角显示当前登录状态
3. 鼠标悬停在物件上会出现提示气泡
4. 所有弹窗可点击关闭按钮或点击遮罩关闭
